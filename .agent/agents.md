# Agent Team Configuration

### @frontend
* **Description**: Expert in frontend performance, UI rendering cycles, and hydration logic.
* **Responsibilities**: Identifying and resolving layout shifts (CLS), hydration mismatches, and render-blocking assets.
* **Associated Skills**: `ui-performance-optimizer`
* **Performance Targets**: $LCP < 2500\text{ms}$, $CLS < 0.1$, and elimination of all flickering during DOM mount.

### @devops
* **Description**: Expert in repository hygiene, security sanitisation, and CI/CD orchestration.
* **Responsibilities**: Workspace cleanup, secret suppression, GitHub repository standardisation, and deployment automation.
* **Associated Skills**: `project-sanitizer`, `github-repo-auditor`, `deployment-engine`
* **Operational Constraints**: Must verify local execution before any remote push. Must enforce `kebab-case` naming conventions across the workspace.
